DocuMind is a **Retrieval-Augmented Generation (RAG) PDF chatbot** that lets users upload PDFs and ask questions based strictly on their content. It uses FAISS for semantic search and HuggingFace models for local inference.

## Features
- Upload multiple PDFs
- Context-aware question answering
- FAISS vector store for retrieval
- HuggingFace embeddings & LLM
- Fully local (no API keys required)

## Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- HuggingFace
- PyPDF2
